# Introduction

Welcome! If you want to learn how to code in Roblox using the Luau language, you're in the right place.

This course is split into multiple parts.

- **Luau**\
  This part will teach you everything about Luau.

- **Roblox**\
  This part will teach you about how to work with the Roblox API, and how
  to script Roblox games.
