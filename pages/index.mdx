# Introduction

Welcome! If you want to learn how to code in Roblox using the Luau language, you're in the right place.

This course is split into multiple parts.

- **Luau**\
  This part will teach you everything you need to know about the Luau language,
  which is the language used to script Roblox games.

- **Roblox**\
  This part will teach you about how to work with the Roblox API, this is needed
  in order to make Roblox games.
